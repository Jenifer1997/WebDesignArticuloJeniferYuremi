ARTICULO "DISEÑO WEB RESPONSIVO"

(Responsive Web Design)

Jenifer Yuremi Osoria Garcia

Resumen (Abstract)

A continuación, se hará una breve descripción de cómo es que surgió tanto la técnica, algunas características y beneficios que el diseño web responsivo adopta. El Responsive Design o diseño adaptativo es la técnica que se usa en la actualidad para tener una misma web adaptada a las diferentes plataformas que nos brinda la tecnología: ordenador, tablet, Smartphone, iPad, Kindle y, en resumen, todas las resoluciones de pantalla existentes. Es un concepto de desarrollo web que se centra en hacer que los sitios se vean y se comporten de manera óptima en todos los dispositivos informáticos personales, desde el escritorio hasta el móvil.

Introducción

En los primeros días del diseño web, las páginas se diseñaban para llenar un tamaño de pantalla en particular. Si el usuario tenía una pantalla más grande o más pequeña que la del diseñador, los resultados esperados iban desde barras de desplazamiento no deseadas hasta longitudes de línea excesivamente largas y un mal uso del espacio. A medida que estuvieron disponibles tamaños de pantalla más diversos, apareció el concepto de diseño web responsivo (RWD, responsive web design), un conjunto de prácticas que permite a las páginas web alterar su diseño y apariencia para adaptarse a diferentes anchos de pantalla, resoluciones, etc.

En 2008 se creó la idea y el propósito del diseño web adaptativo, la recomendación puntualizaba que está hecha en el contexto “One Web” y que engloba la navegación en dispositivos móviles como también en dispositivos de mayor resolución de pantalla. Esta idea hace referencia a crear una web para todos, y accesible desde cualquier tipo de dispositivo.

Desarrollo

El Responsive Web Design consiste en una serie de hojas de estilo en CSS3, que usa el atributo “mediaquery” convirtiendo una web ordinaria en una web multiplataforma capaz de adaptarse a todos los tamaños que existen. Prácticamente hace que la única web que se ha construido se vea correctamente y pueda aprovechar las particularidades de todo dispositivo que exista hasta el día de hoy o pueda existir, independientemente de la pantalla en la que se pueda mostrar.

Ha evolucionado conforme pasan los años. Haciendo que las imágenes se puedan ajustar automáticamente. Esta técnica detecta el ancho de pantalla del navegador y adapta automáticamente todos los elementos que tiene, desde las imágenes hasta el menú de inicio y los tamaños de las letras. El contenido y el diseño se adaptan a través de código CSS, por ello, es importante seleccionar el tamaño de los elementos. Ethan Marcotte fue el responsable que acuño el término de Responsive Web Design en un post en “List Apart” y un libro donde explica todas las circunstancias y técnicas relacionadas con el diseño adaptable, fue el primero en usar el término, mas no el primero en pensar en la necesidad de adaptar el layout de las webs a las diferentes pantallas, se dice que los creadores del estándar CSS ya habían creado tecnologías para producir sitios adaptable.

Una forma interesante de trabajar con este diseño es utilizar las propiedades min-width o max-width donde se define tamaños mínimos o máximos para que la pagina pueda ampliar o reducir según sea necesario dependiendo de la pantalla del dispositivo utilizado. Con estas propiedades se pueden crear diseños que aprovechen al máximo toda la pantalla de dispositivos pequeños, e igual se establecen máximos en pantallas de dispositivos grandes, para crear unos espacios visuales que se vean más agradables.

Conclusión

Se entiende que el termino responsive en general, es la característica que tiene respuesta hacia el medio que le rodea. Se encuentra en el día a día. Este tipo de diseño nos permite estar en última tendencia de desarrollo online, ya que adapta el sitio web a cada dispositivo que hay. Se dice que abarca una serie de características y técnicas tanto de CSS y HTML, además hace uso de las mediaquery ya que son las bases de responsive.

Es muy recomendable utilizar el responsive design en una página web ya que ofrece una buena experiencia al usuario y los ayuda a conectar con lo que ofrece dicha página o sitio web adaptándolo a cualquier dispositivo. También se ha vuelto más fácil para lograr diseños receptivos con la ayuda de este método.

A partir se de esta investigación se ha llegado a concluir que el código que utiliza el media querie es fácil y rápido de aprender, ya que, para poder utilizarlo a la hora de querer hacer nuestro sitio adaptable, se necesitan especificar las propiedades que se quieren acoplar, y con ellas el sitio se adecua a las necesidades de los distintos dispositivos.
